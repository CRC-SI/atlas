# Atlas

** Abstract API for Geospatial Rendering **

## About

[More Info](http://urbanetic.net/atlas/)

## Install
```
npm install
grunt install
```

## Build
Atlas can be built into compressed JavaScript and CSS resources:

```
grunt build
```
See `dist/` for the output.

## Docs
Generate JSDocs in `docs/` with:

```
grunt doc
```
